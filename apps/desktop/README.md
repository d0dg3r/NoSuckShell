# Desktop app (`apps/desktop`)

This package is the **NoSuckShell** product: a **Tauri 2** app with a **Vite + React** frontend and a **Rust** backend in `src-tauri/`.

## Commands

Most day-to-day commands are documented at the **repository root** (install, `tauri:dev`, tests, `cargo`):

- [Contributing / setup](../../CONTRIBUTING.md)
- [Architecture](../../docs/architecture.md)
- [Tauri documentation](https://v2.tauri.app/)

From this directory you can still run `npm install`, `npm run dev`, `npm run tauri:dev`, `npm test`, and `npm run build` directly.
